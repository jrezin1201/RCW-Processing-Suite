Contracts-Forms Processing

Transforms construction project task lists into formatted billing summaries. Automatically extracts project name, phase, and house string from input files. Categorizes tasks into EXT PRIME, EXTERIOR, EXTERIOR UA, and INTERIOR buckets. Formats lot numbers, combines plan/elevation codes, and generates professional Excel outputs with project metadata, totals by lot/plan, and automatic labor (43%) and material (28%) calculations. Includes comprehensive QA reporting for unmapped tasks. Supports both .XLS and .XLSX formats.
